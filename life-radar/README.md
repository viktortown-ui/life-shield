# Радар жизненной устойчивости

Рабочий MVP веб-платформы для оценки и отслеживания состояния жизни.

## Структура проекта

```
life-radar/
├── index.html          # Основной HTML файл с разметкой всех экранов
├── styles.css          # CSS стили
├── data.js            # Конфигурация сфер и вопросов
├── calculator.js      # Логика расчета индексов
├── radar.js           # Визуализация радара и графиков
├── app.js             # Основная логика приложения
└── README.md          # Этот файл
```

## Как запустить

### Вариант 1: Просто открыть файл (для тестирования)

1. Откройте файл `index.html` в браузере
2. Готово! Приложение работает

### Вариант 2: Локальный сервер (рекомендуется)

```bash
# Перейти в папку проекта
cd life-radar

# Запустить локальный сервер
python -m http.server 8000

# Открыть в браузере
# http://localhost:8000
```

### Вариант 3: Деплой на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите файлы в репозиторий
3. Включите GitHub Pages в настройках репозитория
4. Приложение будет доступно по ссылке вида: `https://username.github.io/repository-name/`

## Что реализовано

### ✅ Экраны
- **Экран приветствия** — ввод имени, история снимков
- **Экран оценки** — 24 вопроса по 6 сферам жизни
- **Экран весов** — настройка важности сфер
- **Экран радара** — визуализация результатов
- **Экран интерпретации** — зоны и приоритеты
- **Экран истории** — график изменений и таблица

### ✅ Функциональность
- **Сохранение данных** в localStorage
- **Создание снимков** состояния
- **Пересчет индексов** в реальном времени
- **Визуализация радара** с помощью Chart.js
- **График динамики** изменений
- **Экспорт/импорт** данных (можно добавить)

### ✅ Методы расчета
- Нормализация ответов в диапазон 0-1
- Взвешенное среднее по сферам
- Пороговая логика (зеленый/желтый/красный)
- Формирование приоритетов

## Как использовать

1. **Первый запуск**
   - Введите имя или псевдоним
   - Нажмите "Начать новый снимок"

2. **Оценка состояния**
   - Ответьте на 24 вопроса по 6 сферам
   - Используйте шкалы и радиокнопки
   - Вопросы о "сейчас", а не о "вообще"

3. **Настройка весов**
   - Установите важность каждой сферы (1-5)
   - Можно изменить при следующем снимке

4. **Радар**
   - Посмотрите визуализацию результатов
   - Общий индекс и подиндексы по сферам
   - Цветовая индикация состояния

5. **Интерпретация**
   - Стабильные зоны (зеленые)
   - Зоны внимания (желтые)
   - Зоны риска (красные)
   - Направления для внимания

6. **История**
   - График изменений общего индекса
   - Таблица всех снимков
   - Сравнение "было → стало"

7. **Повторное использование**
   - "Новый снимок" — полная переоценка
   - "Продолжить" — быстрая корректировка

## Технические детали

### Хранение данных
- localStorage (до 5MB)
- Формат: JSON с массивом снимков
- Каждый снимок содержит ответы, веса, индексы

### Зависимости
- Chart.js для визуализации (CDN)
- Нет других внешних зависимостей

### Совместимость
- Современные браузеры (Chrome, Firefox, Safari, Edge)
- Мобильные устройства (адаптивная верстка)
- Не требует установки
