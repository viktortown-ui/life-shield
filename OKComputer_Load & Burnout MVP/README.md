# Нагрузка и выгорание

Инженерная модель оценки времени и энергии для понимания устойчивости вашего режима жизни.

## Описание

Это не психологический тест и не медицинский диагноз. Это инструмент для осознания своей загруженности и поиска баланса между нагрузкой и восстановлением.

## Как использовать

### 1. Запуск приложения

Откройте файл `index.html` в браузере или запустите локальный сервер:

```bash
python -m http.server 8000
```

### 2. Экран 1 - Распределение времени

Введите количество часов в неделю, которые вы тратите на:
- Сон
- Работа
- Проекты
- Быт
- Семья / Общение
- Восстановление (спорт / отдых)

Система автоматически покажет общее количество часов и предупредит о перерасходе (>168 часов в неделю).

### 3. Экран 2 - Энергия и стресс

Оцените по шкале от 1 до 5:
- Субъективная энергия (как вы себя чувствуете последние 2-3 недели)
- Уровень хронического стресса

### 4. Экран 3 - Нагрузка

Система рассчитает:
- Общий индекс нагрузки
- Индекс восстановления
- Коэффициент перегруза

### 5. Экран 4 - Риск выгорания

На основе всех данных система покажет:
- Светофор риска (зелёный / жёлтый / красный)
- Процент риска выгорания
- Детали расчёта

### 6. Экран 5 - История

Сохранённые результаты с возможностью сравнения "было → стало".

## Методика расчёта

### Коэффициенты нагрузки
- Работа: 1.2 (высокая нагрузка)
- Проекты: 1.1 (высокая нагрузка)
- Быт: 0.8 (средняя нагрузка)
- Семья: 0.7 (средняя нагрузка)
- Восстановление: -0.5 (уменьшает нагрузку)
- Сон: -1.0 (сильно уменьшает нагрузку)

### Факторы риска
1. Перегруз по времени (превышение 168 часов в неделю)
2. Дисбаланс энергии (низкая оценка энергии)
3. Недостаток восстановления (коэффициент перегруза > 1)

### Уровни риска
- **Зелёный (< 30%)**: Режим устойчив
- **Жёлтый (30-60%)**: Граница
- **Красный (> 60%)**: Выгорание почти гарантировано

## Важно понимать

⚠️ **Ограничения модели:**

1. Эта модель упрощённая и не учитывает множество индивидуальных факторов
2. Все данные субъективны и зависят от вашей самооценки
3. Это не медицинский диагноз и не заменяет консультацию специалиста
